# CRC-CompanionRapportCooldown
An Addon-Project for TESO

Started on May, 15 2026 by Cudddel with inspirations by Claude

If project is finished this Addon will check cooldown für rapportgain of The Elder Scolls online Companions.

Planned features:

- Handle known cooldowns per companions and toon
- Learn unknown cooldowns just by playing the game
- Reminder if cooldown has ended 
- Open for upcoming campanions

What it not provides:

- Handle rapportlost
- Block "forbidden" actions
